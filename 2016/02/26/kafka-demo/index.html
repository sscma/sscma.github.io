kafka demo

  • big data

最近公司有个实时日志分析，准备采用kafka，自己在windows机器上尝试搭建了一下，下面是搭建步骤
1.zookeeper集群为了方便就一台机器 localhost:2181
2.下载kafka，记得下载带scala的预编译版本
https://www.apache.org/dyn/closer.cgi?path=/kafka/0.9.0.1/kafka_2.11-0.9.0.1.tgz
3.解压，并重命名为多个文件夹
4.修改配置文件，目前只需要需改的地方有3个，假设kafka集群机器有两台机器，则server.properties需要修改如下3个地方
（1）broker.id=0
listeners=PLAINTEXT://:9092
log.dirs=/tmp0/kafka-logs
（2）broker.id=1
listeners=PLAINTEXT://:9093
log.dirs=/tmp1/kafka-logs
5.启动，先启动zookeeper，后启动两台kafka
kafka的启动需要到bin/windows目录下，运行如下命令
kafka-server-start.bat ../../config/server.properties
6.测试，同样需要进入bin/windows目录下，然后运行命令
（1）创建topic
kafka-topics.bat -zookeeper localhost:2181 -topic test -replication-factor 2 -partitions 5 -create
（2）查看topic
kafka-topics.bat -zookeeper localhost:2181 -list
（3）创建producer
kafka-console-producer.bat -broker-list localhost:9092,localhost:9093 -topic test
（4）创建consumer
kafka-console-consumer.bat -zookeeper localhost:2181 - from-begining -topic test
（5）发送消息
